<h1>Visa Check Netherlands 2026 — Which Dutch Visa or Permit Do You Need?</h1>
<p>Moving to the Netherlands is one of the most popular decisions for international professionals, students, and families in Europe. The Dutch immigration system, managed by the IND (Immigratie- en Naturalisatiedienst), offers a range of visa and residence permit options tailored to different circumstances. Understanding which permit applies to your situation is the critical first step — the wrong application can cost months of processing time and significant fees.</p>

<h2>Overview of the Dutch Immigration System</h2>
<p>The Netherlands uses a two-document system for most non-EU immigrants. An <strong>MVV (Machtiging tot Voorlopig Verblijf)</strong> is a long-stay entry visa issued by the Dutch embassy in your home country. It is valid for 90 days and allows you to enter the Netherlands to collect your actual residence permit. The <strong>VVR (Verblijfsvergunning Regulier)</strong> is the residence permit card that confirms your right to stay, work, or study in the Netherlands for an extended period.</p>
<p>Not all permit types require both documents. For example, the Highly Skilled Migrant permit and the EU Blue Card can be applied for by the employer directly in the Netherlands before the employee arrives, streamlining the process significantly. The IND (Immigration and Naturalisation Service) is the government body that processes all applications.</p>

<div class="info-box">
<strong>Key tip:</strong> The single most important factor in Dutch immigration is whether your employer is an IND "recognised sponsor." Recognised sponsors can apply on your behalf and benefit from faster processing (2-4 weeks vs. months). You can check if your employer is listed on the <a href="https://ind.nl/en/public-register-recognised-sponsors" target="_blank" rel="noopener noreferrer">IND public register</a>.

<h2>EU vs Non-EU: The Critical Distinction</h2>
<p>If you hold a passport from an EU or EEA country (Norway, Iceland, Liechtenstein) or Switzerland, Dutch immigration is straightforward. You simply register at your local municipality (gemeente) within four months of arrival. There are no visa applications, no IND fees, and no salary thresholds. You have the right to live, work, study, and start a business in the Netherlands without restriction.</p>
<p>For non-EU nationals, the process is more structured. You must identify the correct permit category, meet the specific requirements (salary thresholds, employer recognition, educational enrolment, family status), and apply through the IND. The fastest route is always through a recognised sponsor employer using the Highly Skilled Migrant track.</p>

<h2>The Highly Skilled Migrant Route Explained</h2>
<p>The Kennismigrant (Highly Skilled Migrant) permit is the most common route for international professionals working in the Netherlands. To qualify, you need a concrete job offer from a company that is registered as an IND recognised sponsor — thousands of Dutch companies, from startups to multinationals, have this status. Your gross monthly salary must exceed the IND thresholds (updated annually each January): roughly €4,752/month for those aged 30 and over, and €3,484/month for those under 30 in 2025 figures.</p>
<p>The IND processes Kennismigrant applications within 2-4 weeks when submitted by the employer. Your 30% tax ruling application can be submitted simultaneously, potentially saving you thousands of euros per year by allowing 30% of your salary to be paid tax-free for up to five years.</p>

<h2>Starting a Business or Working Freelance</h2>
<p>Non-EU nationals who want to work as a freelancer or start a company in the Netherlands need a self-employment residence permit. Unlike employment permits, there is no recognised sponsor: the IND assesses your business plan on a points-based system evaluating personal experience, the economic value of your work to the Netherlands, and the business plan quality. Registering with the Dutch Chamber of Commerce (KVK) is mandatory. The process takes 8-12 weeks.</p>
<p>If you are being transferred from your foreign employer to a Dutch branch or subsidiary, the Intra-Corporate Transfer (ICT) permit is faster and requires fewer documents — your employer must simply confirm you have been employed for at least 3 months and are filling a manager or specialist role.</p>

<h2>Common Mistakes in Dutch Visa Applications</h2>
<ul>
<li><strong>Applying in the wrong category</strong> — e.g., applying for a tourist visa when you intend to work. Even a single day of work on a Schengen visa is illegal and can result in future applications being denied.</li>
<li><strong>Not verifying employer recognition</strong> — the Kennismigrant route is only available through a recognised sponsor. Check the IND public register before accepting a job offer.</li>
<li><strong>Underestimating processing times</strong> — especially for family reunification (8-12 weeks) and self-employment permits. Plan well ahead of your planned move date.</li>
<li><strong>Forgetting the MVV step</strong> — many permit types require you to apply for an MVV entry visa at a Dutch embassy before travelling, not after arrival.</li>
<li><strong>Missing the income requirement for family reunification</strong> — the sponsor (your partner or family member in NL) must earn at least 100% of the Dutch minimum wage to bring you over.</li>
<li><strong>Not registering at the municipality</strong> — even after receiving your residence permit, you must register at your local gemeente (municipality) within 5 days of arrival to receive your BSN (citizen service number), which is required for banking, healthcare, and employment.</li>
</ul>

currentStep: 1,

const TOTAL_STEPS = 5;
const STEP_LABELS = ['Nationality', 'Purpose', 'Details', 'Extra Info', 'Result'];

/* ===================== STEP INDICATOR ===================== */
function renderIndicator() {
const el = document.getElementById('stepIndicator');
let html = '';
for (let i = 1; i <= TOTAL_STEPS; i++) {
const isCompleted = i < state.currentStep;
const isActive = i === state.currentStep;
html += `<div class="step-item-wrap">
<div style="display:flex;align-items:center;width:100%;">
${i > 1 ? `<div class="step-line${isCompleted ? ' completed' : ''}"></div>` : ''}
<div class="step-circle${isCompleted ? ' completed' : isActive ? ' active' : ''}">
${isCompleted ? '&#10003;' : i}
</div>
${i < TOTAL_STEPS ? '<div class="step-line' + (isCompleted ? ' completed' : '') + '"></div>' : ''}
</div>
<span class="step-label" style="color:${isActive ? '#E8832A' : isCompleted ? '#1B6B4A' : '#9CA3AF'};font-weight:${isActive || isCompleted ? '600' : '400'}">${STEP_LABELS[i-1]}</span>
</div>`;
}
el.innerHTML = html;
}

/* ===================== RADIO GROUPS ===================== */
function initRadioGroups() {
document.querySelectorAll('.radio-opt').forEach(opt => {
opt.addEventListener('click', function() {
const group = this.dataset.group;
const value = this.dataset.value;
const stateKey = groupToStateKey(group);
if (stateKey) state[stateKey] = value;

document.querySelectorAll(`.radio-opt[data-group="${group}"]`).forEach(o => o.classList.remove('selected'));
this.classList.add('selected');

onSelectionChange(group, value);
});
});

// Salary select
document.getElementById('salarySelect').addEventListener('change', function() {
state.salary = this.value || null;
});

// Toggles
document.getElementById('livedBefore').addEventListener('change', function() { state.livedBefore = this.checked; });
document.getElementById('thirtyPct').addEventListener('change', function() { state.thirtyPct = this.checked; });
document.getElementById('blueCard').addEventListener('change', function() { state.blueCard = this.checked; });
document.getElementById('orientation').addEventListener('change', function() { state.orientation = this.checked; });
}

function groupToStateKey(group) {
const map = {
euGroup: 'eu', purposeGroup: 'purpose',
jobOfferGroup: 'jobOffer', sponsorGroup: 'sponsor', ageGroup: 'age',
bizTypeGroup: 'bizType', enrolledGroup: 'enrolled', durationGroup: 'duration',
joiningGroup: 'joining', marriedGroup: 'married'
};
return map[group] || null;
}

function onSelectionChange(group, value) {
if (group === 'purposeGroup') showPurposeSection(value);
}

function showPurposeSection(purpose) {
['workSection','businessSection','studySection','familySection','retireSection','otherSection'].forEach(id => {
document.getElementById(id).classList.remove('visible');
});
const sectionMap = {
work: 'workSection', business: 'businessSection', study: 'studySection',
family: 'familySection', retire: 'retireSection', other: 'otherSection'
};
if (sectionMap[purpose]) document.getElementById(sectionMap[purpose]).classList.add('visible');
}

/* ===================== NAVIGATION ===================== */
function showStep(n) {
document.querySelectorAll('.step-panel').forEach(p => p.classList.remove('active'));
document.getElementById('step' + n).classList.add('active');
state.currentStep = n;
renderIndicator();
window.scrollTo({ top: document.getElementById('wizardCard').offsetTop - 80, behavior: 'smooth' });
}

function goNext(currentStep) {
if (!validateStep(currentStep)) return;

// Step 1 shortcut: EU citizen → skip to result
if (currentStep === 1 && state.eu === 'yes') {
showResultPanel('EU');
showStep(5);
return;
}

// Step 2: set up step 4 toggles based on purpose
if (currentStep === 2) {
document.getElementById('thirtyPctRow').style.display = state.purpose === 'work' ? 'flex' : 'none';
document.getElementById('blueCardRow').style.display = state.purpose === 'work' ? 'flex' : 'none';
document.getElementById('orientationRow').style.display = (state.purpose === 'work' || state.purpose === 'other') ? 'flex' : 'none';
}

showStep(currentStep + 1);

// Track step completion
if (typeof gtag !== 'undefined') {
gtag('event', 'wizard_step', { step: currentStep + 1, tool: 'visa_finder' });
}
}

function goPrev(currentStep) {
// If we're at step 3 and EU was yes, go back to step 1
if (currentStep === 5 && state.eu === 'yes') {
showStep(1);
return;
}
showStep(currentStep - 1);
}

function validateStep(step) {
if (step === 1 && !state.eu) {
alert('Please select your nationality status to continue.');
return false;
}
if (step === 2 && !state.purpose) {
alert('Please select your purpose of stay to continue.');
return false;
}
return true;
}

/* ===================== DECISION TREE ===================== */
function decideVisa() {
const { eu, purpose, jobOffer, sponsor, salary, age, bizType, enrolled, duration, joining, orientation } = state;

// EU citizen
if (eu === 'yes') return 'EU';

// Short stay / other
if (purpose === 'other') {
if (orientation) return 'Orientation';
return 'ShortStay';
}

// Retire
if (purpose === 'retire') return 'Retirement';

// Study
if (purpose === 'study') {
if (duration === 'short') return 'ShortStay';
return 'Student';
}

// Family
if (purpose === 'family') return 'Family';

// Business
if (purpose === 'business') {
if (bizType === 'ict') return 'ICT';
return 'SelfEmployed';
}

// Work
if (purpose === 'work') {
// No job offer yet
if (jobOffer === 'no') {
if (orientation) return 'Orientation';
return 'ShortStay';
}

// Has job offer
if (sponsor === 'yes') {
// Check for Blue Card preference and high salary
if (state.blueCard && (salary === 'very_high' || salary === 'high')) {
return 'BlueCard';
}
// Standard Kennismigrant
if (salary === 'low') return 'TWV'; // below threshold
return 'HSM';
}

if (sponsor === 'no') {
// Check orientation year as fallback
if (orientation) return 'Orientation';
return 'TWV';
}

// sponsor = 'unsure'
if (salary === 'very_high' || salary === 'high') return 'HSM';
if (salary === 'mid') return age === 'under30' ? 'HSM' : 'TWV';
return 'TWV';
}

return 'ShortStay';
}

/* ===================== RESULT ===================== */
function showResult() {
const visaKey = decideVisa();
showResultPanel(visaKey);
showStep(5);

if (typeof gtag !== 'undefined') {
gtag('event', 'visa_result', { visa_type: visaKey, tool: 'visa_finder' });
}
}

Object.assign(state, {
currentStep: 1, eu: null, purpose: null,
jobOffer: null, sponsor: null, salary: null, age: null,
bizType: null, enrolled: null, duration: null,
joining: null, married: null,
livedBefore: false, thirtyPct: false, blueCard: false, orientation: false
});

document.querySelectorAll('.radio-opt').forEach(o => o.classList.remove('selected'));

// Clear select
document.getElementById('salarySelect').value = '';

// Clear toggles
['livedBefore','thirtyPct','blueCard','orientation'].forEach(id => {

});

// Hide sub-sections

if (typeof gtag !== 'undefined') {
gtag('event', 'wizard_reset', { tool: 'visa_finder' });
}
}
